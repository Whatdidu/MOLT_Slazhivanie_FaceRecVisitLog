О проекте

Sputnik FaceID — это система автоматического учёта посещений офиса на основе распознавания лиц. Камера фиксирует сотрудников при входе и выходе, система распознаёт их и ведёт журнал посещений.

Проект разработан командой из 4 человек в рамках программы Слаживания. Каждый участник отвечал за свой модуль системы.

Возможности системы

Что умеет Sputnik FaceID

Распознавание лиц

Автоматическое определение сотрудников по лицу с использованием нейросетей dlib

Интеграция с камерой

Камера отправляет снимки по FTP при детекции движения, система обрабатывает их в реальном времени

Журнал посещений

Автоматическая фиксация времени прихода и ухода каждого сотрудника

Управление сотрудниками

Добавление новых сотрудников с фото, редактирование данных, просмотр статуса

Отчёты и аналитика

Статистика посещений, отчёты по сотрудникам, экспорт данных

Веб-админка

Удобный интерфейс для управления системой, защищённый паролем
