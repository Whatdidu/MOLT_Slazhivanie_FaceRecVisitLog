# Scripts

Служебные скрипты для управления проектом.

## init_test_data.py

Инициализирует тестовые данные в БД.

### Использование

```bash
# Запустить после применения миграций
python scripts/init_test_data.py
```

### Что создается

- 5 тестовых сотрудников с разными отделами
- Проверяет дубликаты по email
